# Devilicious Treats Automation System

## Overview
Built an AI-powered customer service automation system for a specialty bakery using n8n workflow automation.

## Architecture
- **Frontend:** Facebook Messenger API
- **Backend:** n8n workflow engine  
- **AI:** Google Gemini integration
- **Data:** Google Sheets backend

## Technical Highlights
- Real-time message processing
- AI-driven customer intent recognition
- Rate-limited batch processing
- Multi-platform integration

## Results
- 80% reduction in manual response time
- Automated customer qualification
- Scalable order management system
